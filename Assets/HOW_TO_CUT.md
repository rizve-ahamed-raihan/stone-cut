# 🎯 HOW TO CUT STONES - Simple Instructions

## 🚀 SETUP FIRST (Do This Once)

### 1. Open Unity and Your Scene
- Open `Assets/Scenes/stonecutscenee.unity`

### 2. Run Auto-Setup (ONE CLICK!)
- Go to Unity menu: **Tools → Stone Cutting → 🚀 Quick Game Setup**
- Wait for "Setup Complete!" message in Console
- This creates everything you need automatically!

## ✂️ HOW TO CUT (In Play Mode)

### Step 1: Press Play ▶️
Click the Play button at the top of Unity

### Step 2: Click on a Stone
- Move your mouse over a stone
- Press and HOLD the left mouse button

### Step 3: Drag Across the Stone
- While holding the mouse button, drag across the stone
- You'll see a red line showing where you'll cut

### Step 4: Release to Cut!
- Release the mouse button
- **BOOM!** The stone splits into two pieces! ✨

## 🎮 Example:

```
1. Press Play ▶️
2. Click on stone [HOLD mouse button]
3. Drag -----> [draw line across stone]
```

## 📋 Troubleshooting

### "Nothing happens when I click"
**Solution:** Run the setup first!
```
Tools → Stone Cutting → 🚀 Quick Game Setup
```

### "Stone doesn't cut when I drag"
**Check:**
1. Are you in Play mode? (▶️ button should be blue)
2. Did you run Quick Game Setup?
3. Is there a "Game Manager" in your hierarchy?

**Fix:** Run this in Unity menu:
```
Tools → Stone Cutting → 🔧 Validate Scene
```

### "I see an error"
**Fix:** Run Quick Game Setup again:
```
Tools → Stone Cutting → 🚀 Quick Game Setup
```

## 💡 Pro Tips:

- **Slow drag** = Precise, clean cut
- **Fast drag** = Dynamic, forceful cut
- **Drag at angles** = Cut in different directions
- **Cut pieces again** = Make smaller pieces!

## 🎯 Alternative Controls:

If drag-to-cut doesn't work, try **click-to-cut**:

1. Stop Play mode
2. Select your stone in hierarchy
3. Add component: `SimpleStoneCutter`
4. Assign "Mesh Cutter" in inspector
5. Press Play
6. Just **CLICK** on the stone (no dragging needed)
