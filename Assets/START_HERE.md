## 🎨 Customization

### Change Cut Force:
1. Select "Game Manager" in hierarchy
2. Find "StoneCuttingGame" component
3. Adjust "Cut Force" slider (default: 5)

### Enable Limited Cuts:
1. Select "Game Manager"
2. Uncheck "Unlimited Cuts"
3. Set "Cuts Remaining" (default: 10)

### Add Cutting Tool Visual:
1. Find a 3D model (sword, knife, blade)
2. Assign to "Cutting Tool" field in Game Manager

### Change Stone Models:
1. Delete current stones
2. Drag new models from Assets
3. Run `Setup All Stones`
